# NoomiBodi User Roles

## 1. admin

The application owner/operator role.

- Access to AI usage dashboard
- User management
- Database admin tools
- Can impersonate other users (for support)
- View all analytics across all users

## 2. beta

Pilot users — friends and family testing the app.

- Full app access
- Free during pilot period
- No API key required (subsidized by admin)
- Priority support/feedback channel

## 3. pro

Future paid subscription tier.

- All features
- Higher usage limits
- Advanced analytics/reports
- Premium AI features

## 4. standard

Future basic/free tier.

- Core features
- Usage limits
- Basic reporting

## 5. byok

Bring Your Own Key — users who supply their own Claude API key.

- Full features
- Provide own Claude API key
- No usage limits (user pays their own API costs)
